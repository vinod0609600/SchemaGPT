# AI Text-to-SQL Chatbot

An AI-powered Text-to-SQL chatbot built using Flask, SQLite, and Groq LLM.

## Features

- Natural language to SQL
- Dynamic schema retrieval
- Multi-table querying
- Foreign key support
- Human-like AI responses
- Case-insensitive search
- AI-generated explanations
- Conversational database assistant

## Technologies Used

- Python
- Flask
- SQLite
- Groq API
- HTML/CSS
- Jinja2

## Run Locally

```bash
pip install -r requirements.txt
python app.py
